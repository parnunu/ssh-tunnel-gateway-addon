# Changelog

## 1.0.1 - 2026-04-22

- Added per-tunnel `ssh_private_key` support using files stored under `ssh/keys/`.
- Switched SSH host key handling to automatic managed `known_hosts` storage.
- Clarified setup documentation and first-run behavior.

## 1.0.0 - 2026-04-21

- Initial release.
- Added multi-tunnel SSH local forwarding with automatic reconnect.
- Added persistent key and `known_hosts` handling.
- Added central repository publish workflow.
